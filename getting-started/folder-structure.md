### Folder Descriptions

| Folder / File                       | Purpose                                                                                                   |
| ----------------------------------- | --------------------------------------------------------------------------------------------------------- |
| **config/**                         | Stores credentials, configuration files, and log settings.                                                |
| **config/credentials.yaml**         | Contains your API Key, Secret, and environment details. Keep this file secure and never commit it to Git. |
| **src/**                            | Main source directory for SDK scripts, automation, and integrations.                                      |
| **src/modules/**                    | Custom detection or response logic built on top of CyberShield SDK functions.                             |
| **src/utils/**                      | Utility scripts for data formatting, validation, or API helpers.                                          |
| **logs/**                           | Stores execution logs and error reports generated by SDK modules.                                         |
| **tests/**                          | Contains basic connectivity and health-check test scripts.                                                |
| **requirements.txt / package.json** | Manages SDK dependencies (Python / Node.js).                                                              |
| **.env**                            | Stores environment variables securely for local development.                                              |
| **README.md**                       | Project-level documentation or usage summary.                                                             |
